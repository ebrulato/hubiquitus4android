String org.hubiquitus.hapi.client.HClient.getLastMessages ( String  chid,
int  nbLastMsg 
)

Demands the hserver a list of the last messages saved for a dedicated channel. The publisher must be in the channel’s participants list.

Nominal response : an hCallback with an hResult will be performed when the result is available. If the hResult had status 0, the user should expect to receive n calls to hCallback of type hMessage where n is equal to the number of messages retrieved with nbLastMsg as an upper limit.

Warning:
HResult result type will be a JSonArray if successful
Parameters:
chid- channel id
nbLastMsg
Returns:
request id
